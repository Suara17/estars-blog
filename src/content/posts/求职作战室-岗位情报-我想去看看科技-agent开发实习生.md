# 我想去看看科技-agent开发实习生

**公司：** 我想去看看（北京）科技有限责任公司  
**融资阶段：** 天使轮 · 20-99人 · 互联网  
**薪资待遇：** 250-300元/天  
**工作地点：** 北京 · 朝阳区 · 鸿运大厦(酒仙桥中路)8层  
**实习周期：** 4天/周 · 3个月 · 本科  
**来源平台：** BOSS直聘  
**提取时间：** 2026-06-14

---

## 你会做什么

直接参与真实产品的开发，**不是写 demo，不是打杂**——代码会跑在生产环境里。

- 独立完成功能模块的**前后端开发**，包括页面、接口、数据库的完整链路
- 参与 **API 设计**与实现，完成标准的 CRUD 功能开发
- 配合 AI 工程师完成与 **Agent 相关的前后端接口对接**
- 参与 **Code Review**，养成规范的工程习惯
- 遇到问题先自己查、自己试，实在搞不定再问

## 我们希望你

### 必须
- 计算机相关专业在读（本科或研究生），计算机基础扎实——理解 HTTP 请求响应、数据库读写、前端渲染的基本原理，不只是会调包
- 能独立写出可运行的前后端 CRUD——有完整的 side project 或课程项目可以展示，哪怕很简单
- 熟悉至少一种后端语言（**Python / Go / Node.js** 均可），能写基础的 REST API
- 熟悉基础 **SQL**，能独立完成常规的增删改查和简单联表
- 熟悉前端基础（HTML / CSS / JavaScript），了解 **React 或 Vue** 其中之一
- 能用 **Git** 进行基本的版本管理，理解 branch / commit / PR 流程
- 能通过 **AI 工具（Cursor / Claude Code 等）** 提升开发效率，但能判断 AI 生成代码的对错，不盲目粘贴
- 每周能保证至少 **4天** 实习，能坚持 **3个月** 以上

### 加分
- 有完整部署上线过的个人项目（哪怕用 Vercel / Railway 等托管平台）
- 了解 **Docker** 基础，能看懂 Dockerfile
- 接触过 **LLM API**（OpenAI / Claude 等），做过任何 AI 相关的小项目
- 熟悉 **Next.js** 或 TypeScript
- 有开源贡献或活跃的 GitHub

> ⚡ **关联分析：** 这个岗位虽然冠以"agent开发"之名，但实际上是一个**全栈工程师+Agent接口对接**的复合型实习。特点是真实生产环境、代码要上线、强调工程师规范。支持AI工具提效但不盲从，跟心尺科技的"全栈+AI应用"方向互补。天使轮公司，团队小，能接触的面会更广。
