## 岗位职责

1. 使用 Vibe Coding 完成全栈产品研发，从需求到上线快速迭代
2. 配合产品参与 AI 工具链、自动化流程、Agent 逻辑搭建与优化
3. 负责AI应用相关后端服务的开发、调试与优化，参与需求分析、技术方案讨论等环节
4. 协助对接AI模型接口，完成模型调用、数据交互等开发工作，并与前端完成联调测试
5. 收集开发过程中的问题，整理技术文档，配合团队完成项目迭代与优化

## 任职要求

1. 计算机科学、人工智能相关专业本科及以上学历
2. 精通 Vibe Coding，熟练使用 Claude Code、Cursor 等 AI Coding 工具；熟悉 Python 编程语言，具备扎实的编程基础，了解常用的数据结构、算法及设计模式
3. 了解后端开发流程，熟悉HTTP/HTTPS协议、并发编程、数据库等相关技术
4. 了解AI相关基础知识，具有AI模型接口调用、AI应用开发相关实践经验者优先
