---# 🎬 小红书视频下载指南

> 适用场景：从小红书分享链接下载视频（含高清截图提取）
> 最后更新：2026-06-20

---

## 📋 前置条件

| 项目 | 说明 |
|:---|:---|
| 所需工具包 | `extended_http_tools`（发起HTTP请求） |
| 辅助工具 | `ffmpeg`（截取视频帧） |
| 核心思路 | 从小红书网页版HTML源码中提取 `og:video` 标签的视频直链 |

---

## 🚀 操作流程（四步走）

### Step 1️⃣ 获取网页源码

用桌面版UA访问小红书分享链接，获取完整HTML：

```
extended_http_tools:http_request
├── url = 小红书分享链接（xhslink短链）
├── method = GET
└── headers = { "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 ..." }
```

> ⚠️ **关键**：一定要用桌面版 Chrome UA！移动端UA返回的页面可能不含视频链接。
> ⚠️ 响应内容较大时会被保存到文件，用 `grep_code` 检索即可。

---

### Step 2️⃣ 提取视频直链

在HTML中搜索 `og:video` 标签：

```
grep_code
├── path = 上一步保存的HTML文件路径
└── pattern = og:video
```

找到形如以下的行，复制 `content` 属性值：

```html
<meta name="og:video" content="https://sns-video-v3.xhscdn.com/stream/xxx/xxx.mp4?sign=xxx&t=xxx">
```

> 💡 **小贴士**：同一位置附近还有 `og:videotime`（视频时长）、`og:videoquality`（画质）等元信息，可一并参考。

---

### Step 3️⃣ 下载视频

用 `download_file` 下载，**必须带上 Referer 请求头**：

```
download_file
├── url = 上一步提取的视频直链
├── destination = 你想保存的路径（如 /sdcard/Download/xxx.mp4）
└── headers = { "Referer": "https://www.xiaohongshu.com", "User-Agent": "..." }
```

> ⚠️ **重要**：小红书视频CDN有防盗链，不加 `Referer: https://www.xiaohongshu.com` 会下载失败！

---

### Step 4️⃣ 截取视频帧（可选）

下载完成后，用 FFmpeg 截取精彩画面：

```
ffmpeg:ffmpeg_execute
└── command = -i 视频路径 -ss 时间点 -vframes 1 输出图片路径 -y
```

**批量截帧技巧**：
- 先查看视频时长（用 `ffmpeg_info` 或从 `og:videotime` 获取）
- 按时间均匀分布截取点，例如22秒的视频每3秒截一张
- 5秒的短视频每0.5秒截一张，覆盖全部画面
- 输出格式推荐 `.jpg`，画质和体积平衡

示例（截取第3秒的画面）：
```
ffmpeg:ffmpeg_execute
└── command = -i /sdcard/Download/video.mp4 -ss 00:00:03 -vframes 1 /sdcard/Download/frame_01.jpg -y
```

> 🔍 视频信息参考：小红书视频通常为 720×1280，H.264 编码，24fps 或 2fps。

## 📌 常见问题 & 避坑指南

### ❌ 下载失败 / 403 Forbidden
- **原因**：缺少 Referer 或 User-Agent 不对
- **解决**：请求头加上 `Referer: https://www.xiaohongshu.com`，UA 用桌面版

### ❌ 视频链接找不到
- **原因**：网页版未登录时部分内容受限，或链接不是标准笔记页
- **解决**：先用 `visit_web` 访问短链获取真实跳转URL，再对该URL发起HTTP请求

### ❌ 截帧全是黑色/空白
- **原因**：视频有DRM保护或使用了特殊渲染
- **解决**：确认是普通视频笔记（非直播回放、非Vlog特殊格式），普通AI设计视频都可以正常截帧

### ❌ og:video 标签不存在
- **原因**：该笔记是"图文笔记"而非"视频笔记"
- **解决**：图文笔记没有视频链接，只能保存其中的图片（从页面的 `og:image` 获取）

## 🎯 完整操作示例（实战模板）

```
1️⃣ 获取HTML源码
   http_request → GET 链接（桌面UA）

2️⃣ 搜索视频链接
   grep_code → pattern="og:video" → 得到.mp4直链

3️⃣ 下载视频
   download_file → url=直链 + headers={Referer}

4️⃣ 截取视频帧
   ffmpeg_execute → -i 视频 -ss 时间 -vframes 1 输出图片
```
