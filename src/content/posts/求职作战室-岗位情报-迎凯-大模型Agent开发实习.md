# 迎凯-大模型Agent开发实习（提供住宿）

**公司：** 深圳迎凯生物科技有限公司  
**融资阶段：** 天使轮 · 100-499人 · 医疗器械  
**薪资待遇：** 150-300元/天  
**工作地点：** 深圳 · 南太云创谷5栋11、12楼  
**实习周期：** 5天/周 · 6个月 · 本科  
**来源平台：** BOSS直聘  
**提取时间：** 2026-06-14

---

## 岗位职责

1. 参与**大模型智能体（Agent）与技能体系**的开发工作，协助完成技能封装、工具调用、**Function Calling 及任务编排**等功能的实现与测试。
2. 基于 **RAGFlow** 主流 Agent 框架，在导师指导下进行智能体流程设计、**多轮对话逻辑**、规划与执行机制的开发调试与效果验证。
3. 负责 **Prompt 工程**的优化与实验，跟进结构化输出效果，参与交互逻辑的迭代优化。
4. 协助大模型能力与**视觉 OCR、业务系统**的联动开发，完成接口对接与集成测试。
5. 参与功能验证、问题定位、效果评估，并撰写相关技术文档与测试报告。

## 任职要求

1. **熟练使用 Python**，具备良好的代码编写、调试能力，有工程实现的基础经验。
2. 了解 **LLM 基本原理**，对 Agent 架构、技能系统、工具调用、任务规划等概念有一定认知。
3. 了解至少一种主流大模型 Agent 开发框架（如 **LangChain、LlamaIndex、AutoGen、LangGraph、Dify** 等）；有 **RAGFlow** 使用或部署经验者优先。
4. **学习能力强**，逻辑清晰，责任心强，具备良好的沟通能力和团队协作意识，能够在指导下独立推进任务。

## 公司简介

深圳迎凯生物科技有限公司，天使轮融资阶段，100-499人规模，专注于医疗器械领域。
