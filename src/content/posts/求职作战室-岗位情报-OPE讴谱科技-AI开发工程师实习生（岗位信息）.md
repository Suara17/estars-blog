## 职位概述

正在搭建**前沿智能 Agent 共生生态基座**，赋能多元智能体自主联动协作、供需交易、服务、治理落地及共识共建，打造可自我存续的智能生态网络。

实习生将参与 AI Agent 产品和工程能力建设，在导师带领下接触大模型 API、Agent Workflow、RAG、工具调用、Prompt 调优、任务评测与工程化交付。适合已高频使用 AI 编程工具、愿意快速学习 Agent 系统，并希望把项目能力推进到真实产品环境中的同学。

---

## 岗位职责

1. 协助开发 **AI Agent 基础功能**，包括模型调用、Prompt 模板、工具接入、RAG 检索、任务状态记录等模块
2. 在导师指导下使用 **AI 编程工具**完成小型功能开发、Bug 修复、测试用例补充和技术文档整理
3. 参与 **Agent Workflow** 的实验和验证，记录任务输入、执行链路、失败原因、优化方案和评测结果
4. 协助整理知识库、示例任务、测试数据和 Demo 场景，支持产品原型快速迭代
5. 配合产品、设计和研发同学验证 AI 原生功能，将抽象需求拆成可运行、可测试的小任务
6. 跟进主流 **LLM、Agent 框架和 AI 编程工具**的新能力，输出简短调研和可复用实践

## 任职要求

### 0. Vibe Coding / Agentic Coding（必需）
AI 编程工具**重度用户**，熟悉 Codex、Cursor、Claude Code、GitHub Copilot、Windsurf 等至少一种工具。

### 1. 学历背景
本科或硕士在读，计算机科学、软件工程、人工智能、数据科学、数学或相关专业优先。

### 2. 经验要求
可无全职工作经验，但需有 AI 应用、后端开发、个人工具、课程项目、开源项目或实习项目经历。有 LLM API、RAG、Agent、自动化脚本、知识库、工作流工具相关实践优先。

### 3. 核心技能栈
| 维度 | 要求 |
|------|------|
| **编程基础** | 熟悉 Python，了解 JavaScript/TypeScript 更佳，具备 Git 协作能力 |
| **模型调用** | 了解 OpenAI / Claude / Llama 等主流模型 API，能编写并迭代 Prompt |
| **Agent 入门** | 了解 Function Calling、RAG、Workflow、工具调用、上下文管理等基本概念 |
| **工程习惯** | 能写清楚任务记录、测试步骤、复现路径和技术文档 |

### 4. 加分项
- 有 LangChain、LangGraph、CrewAI、OpenAI Agents SDK 或同类框架使用经验
- 有 GitHub 主页、个人 AI 工具、开源贡献、技术博客或长期项目记录
- 熟悉 Obsidian、Markdown、知识库整理、自动化工作流或 AI 编程实践
- 对 AI Agent、智能体协作、AI 原生产品和未来工作方式有持续兴趣
