# 平方和投资-AI Agent开发实习生

**公司：** 宁波平方和投资管理合伙企业（有限合伙）
**融资阶段：** 未融资 · 20-99人 · 基金
**薪资待遇：** 200-400元/天
**工作地点：** 北京 · 海淀区 · 鼎好DH3A座-西南门3层
**实习周期：** 4天/周 · 3个月以上 · 学历不限（本科/研究生在读）
**来源平台：** BOSS直聘
**提取时间：** 2026-06-14

---

## 岗位职责

1. 参与 **AI Agent 应用开发**，围绕办公自动化、ERP、企业流程自动化等场景设计和实现智能 Agent 能力。
2. 基于 **LangChain** 等框架进行 Agent 工具调用、任务规划、上下文管理、工作流集成等能力开发。
3. 参与 **Multi-Agent 系统编排**，包括角色拆分、任务分发、协作流程设计与结果汇总。
4. 参与 **A2A 通讯机制**设计与实现，支持 Agent 间消息传递、任务协同、状态同步与异常处理。
5. 参与 **OpenClaw** 相关能力建设，包括 Agent 工具接入、流程编排、自动化执行与场景验证。
6. 建设和维护 **Agent 评测体系**，包括任务集设计、自动化评测脚本、效果指标定义、失败案例分析等。
7. 结合办公自动化与 ERP 场景，探索 AI Agent 在**审批、报表、数据查询、流程流转、知识检索**等业务中的应用。
8. 跟踪 Agent、LLM 应用开发、多 Agent 协作等方向的前沿技术，并输出调研或原型验证。

## 任职要求

### 必备
1. 计算机、软件工程、人工智能、自动化、信息管理等相关专业本科或研究生在读。
2. 熟悉 **Python**，具备基础的软件工程能力，能够独立完成模块开发、调试和文档整理。
3. 了解 **LangChain、LLM 应用开发、RAG、工具调用、Function Calling** 等相关技术。
4. 对 Agent 架构、任务规划、多轮对话、工具编排、自动化执行有兴趣或实践经验。
5. 了解 **Multi-Agent 协作、A2A 通讯、工作流编排、消息队列**等相关概念者优先。
6. 有办公自动化、ERP、企业系统集成、流程引擎、数据分析等相关项目经验者优先。
7. 具备良好的问题拆解能力、学习能力和工程实现意识，能够主动推进任务落地。
8. 每周可实习 **4天及以上**，可连续实习 **3个月以上**。

### 加分项
1. 使用过 **LangChain、LangGraph、AutoGen、CrewAI、OpenClaw** 或类似 Agent 框架。
2. 有 LLM 应用评测经验，熟悉准确率、任务完成率、工具调用成功率、人工评分、回归评测等指标。
3. 熟悉 API 集成、企业系统接口、**OA/ERP/CRM** 等业务系统。
4. 有开源项目贡献、Agent Demo、自动化脚本、企业流程工具开发经验。
5. 了解 **A2A、MCP**、消息协议、工作流引擎或分布式任务调度。
