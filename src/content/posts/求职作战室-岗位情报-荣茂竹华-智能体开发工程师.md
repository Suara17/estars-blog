## 关于团队

我们是一支处于高速发展阶段的创业团队，正致力于探索和构建下一代 AI Agent 智能体产品。在这里，你将不是一颗螺丝钉，而是核心成员之一。我们提供优厚且具有竞争力的薪资待遇，你将与我们共同经历产品从0到1的完整旅程，将最前沿的AI技术转化为颠覆性的用户体验，并定义智能体与人类协作的未来。

## 核心职责

1. 基于 Hi Agent 平台，功能开发与编码实现（如接口开发、外部系统集成）
2. 搭建并维护智能体知识库，优化知识检索效率，提升智能体决策能力
3. 结合大语言模型（LLM），调优智能体性能，改善用户体验
4. 支持智能体稳定运行，支撑运营数据分析

## 任职要求

1. 本科及以上学历，计算机、人工智能、软件工程等相关专业
2. 1年以上智能体/AI项目开发经验，有 Hi Agent 或类似平台经验者优先
3. 熟练掌握 Python/Java 等一种语言，懂 LLM 原理与提示词工程，熟悉向量数据库
4. 具备问题解决能力、团队协作精神，能承受项目压力
