# 🥇 第一梯队：必备核心技能（频次极高 + 市场价值最高）

这些技能出现在 **80%以上** 的岗位要求中，且往往是**硬性门槛**或**核心职责**，薪资溢价明显。

---

## 1️⃣ 🐍 Python —— 绝对的第一语言

**出现频次：★★★★★（几乎100%岗位要求）**
**价值评级：★★★★★（入门门槛，无可替代）**

| 维度 | 说明 |
|------|------|
| 具体诉求 | Python熟练、Python编程基础、精通Python |
| 岗位示例 | 几乎所有岗位（40+/40+） |
| 辅助技能 | FastAPI/Flask/Django 后端框架（约30%岗位同时要求） |
| 为什么排第一 | 这是所有AI开发岗的基础设施语言，没它后面都不用谈。从低到高、从实习到大厂高薪，Python都是绝对底线。 |
| 💡 建议 | ✅ **必须精通**，不仅是语法，要能写工程级代码、异步编程、FastAPI 后端 |

---

## 2️⃣ 🤖 Agent / 智能体开发 —— 市场的绝对主旋律

**出现频次：★★★★★（约85%岗位直接相关）**
**价值评级：★★★★★（赛道核心，薪资上限最高）**

| 维度 | 说明 |
|------|------|
| 关键词 | AI Agent、智能体、Agent开发、Agent架构 |
| 高频职责 | Agent设计开发、功能模块搭建、工具调用、任务规划、多轮对话 |
| 薪资体现 | 实习岗 200-400元/天（平方和投资），全职岗 15-80K（上海某电商50-80K） |
| 为什么排第二 | Agent是这波AI应用落地的核心载体，几乎所有岗位都围着它转。不要求Agent的岗位要么传统AI（如工业视觉），要么方向特殊。 |
| 💡 建议 | ✅ **必须深度掌握**，从单Agent到多Agent，从demo到工程化 |

---

## 3️⃣ 📡 大模型 / LLM 调用与API集成

**出现频次：★★★★★（约80%岗位要求）**
**价值评级：★★★★★（核心能力，直接决定业务能否落地）**

| 维度 | 说明 |
|------|------|
| 具体技能 | 调用OpenAI/Claude/国产模型API、流式输出、多模型适配 |
| 常见场景 | 模型调用、API接入、接口封装、成本优化、模型对比 |
| 相关高层技能 | 模型能力边界理解、响应延迟优化、成本控制 |
| 💡 建议 | ✅ API调用是基本功，至少要熟悉2-3家模型；能讲清各模型差异和选型逻辑 |

---

## 4️⃣ 🔍 RAG（检索增强生成）

**出现频次：★★★★☆（约70%岗位明确提及）**
**价值评级：★★★★★（解决LLM幻觉问题的关键，大量岗位卡在这）**

| 维度 | 说明 |
|------|------|
| 涉及层级 | 数据预处理 → Embedding → 向量检索 → 重排序 → 与LLM结合 |
| 配套技能 | 向量数据库（Chroma/Milvus/Pinecone/Weaviate）、Embedding模型、分块策略 |
| 岗位举例 | 思灵机器人、中科软、天科数创、奕信科技、青蓝致远……几乎Agent岗都涉及 |
| 💡 建议 | ✅ **必须掌握**，建议至少完整搭过一条RAG链路，踩过Embedding选择、chunk策略、检索效果的坑 |

---

## 5️⃣ ⛓️ LangChain / LangGraph / LlamaIndex 等框架

**出现频次：★★★★☆（约65%岗位明确提及）**
**价值评级：★★★★☆（Agent开发的实际工程工具，求职硬通货）**

| 框架 | 出现场景 | 相关岗位 |
|------|---------|---------|
| **LangChain** | Agent框架、工具调用、RAG管道 | 天科数创、开元云、微聚科技、青蓝致远、龙远鼎晟等 |
| **LangGraph** | 工作流编排、多步推理、状态管理 | 平方和投资、荣联汇智、龙远鼎晟等 |
| **LlamaIndex** | 数据索引、RAG优化 | 数坤科技、奕信科技、开元云、青蓝致远 |
| AutoGen | 多Agent协作 | 唯犀互动、平方和投资、微聚科技、龙远鼎晟 |
| Dify/Coze | 低代码Agent平台 | 唯犀互动、开元云、轩程科技、职由 |
| CrewAI | 多Agent团队协作 | OPE讴谱、天科数创、平方和投资 |
| 💡 建议 | ✅ LangChain是基础入场券，LangGraph是差异化优势，AutoGen/CrewAI是加分项 |

# 🥈 第二梯队：高频刚需技能（频次高 + 有明显价值）

这些技能出现在 **40-65%** 的岗位中，往往是"优先考虑"或"加分项"，但在某些岗位中是硬性要求。

---

## 6️⃣ ✍️ Prompt Engineering / 提示词工程

**出现频次：★★★★（约55%岗位明确提及）**
**价值评级：★★★★（看似基础，但深度拉开差距）**

| 具体诉求 | 频次 |
|---------|------|
| Prompt优化/调优 | 极高 |
| 结构化Prompt设计 | 中 |
| Prompt版本管理 | 较低但高价值 |
| 角色设定/思维链 | 中 |

**一句话总结：** 几乎所有AI岗位都默认你会Prompt，但能把Prompt做出工程化（版本控制、自动化评估、结构化输出）的人才是稀缺的。

---

## 7️⃣ 🔧 Function Calling / 工具调用

**出现频次：★★★★（约50%岗位明确提及）**
**价值评级：★★★★（Agent与真实世界交互的桥梁）**

| 涉及内容 | 说明 |
|---------|------|
| Function Calling原理 | 模型调用外部工具的标准方式 |
| 工具定义/Schema设计 | 定义可被LLM调用的工具接口 |
| 工具链编排 | 多个工具的组合调用、错误处理、回退机制 |
| MCP协议（进阶） | 标准化工具/资源/提示的协议（约15%岗位提及） |

---

## 8️⃣ 💻 Vibe Coding / AI Coding工具使用

**出现频次：★★★☆（约45%岗位提及，但趋势极速上升）**
**价值评级：★★★★（2026年所有科技岗的潜规则）**

| 工具 | 岗位示例 |
|-----|---------|
| Claude Code | OPE讴谱、京东、智谱华章、我想去看看 |
| Cursor | OPE讴谱、京东、我想去看看 |
| Codex | OPE讴谱、心尺科技、智谱华章 |
| GitHub Copilot | OPE讴谱 |
| Windsurf | OPE讴谱 |

**趋势解读：** 不再是"会不会用"，而是"能不能用好"。已经有岗位明确提出"70%样板代码交给AI完成"、"能判断AI生成代码的对错"、"通过TDD+Prompt引导AI写高质量代码"。

---

## 9️⃣ 🗄️ 数据库 / 向量数据库

**出现频次：★★★☆（约45%岗位提及数据库技能）**
**价值评级：★★★★（RAG和业务系统都离不开）**

| 类型 | 具体技术 |
|-----|---------|
| 传统数据库 | MySQL（高频）、Redis（缓存）、MongoDB、PostgreSQL |
| 向量数据库 | Chroma（最常用）、Milvus、Pinecone、Weaviate、FAISS |

---

## 🔟 🏗️ 后端框架能力（FastAPI/Flask/Django）

**出现频次：★★★☆（约40%岗位提及）**
**价值评级：★★★☆（工程化能力证明，Agent部署必须）**

- **FastAPI** 出现最多（异步、高性能、AI项目标配）
- **Flask** 次之（轻量、传统项目）
- **Django** 少数（OPE讴谱等）

# 🥉 第三梯队：加分利器（频次中等 + 特定场景高价值）

---

## 1️⃣1️⃣ 🐳 Docker / 容器化部署

**出现频次：★★★（约35%岗位提及）**
**价值评级：★★★☆（工程化能力的分水岭）**

典型的"不说不知道你会，说了就加分"的技能。不少岗位写的"熟悉Docker操作"、"可Docker部署"。

---

## 1️⃣2️⃣ 🎨 前端开发能力（React/Vue/TypeScript）

**出现频次：★★★（约30%岗位提及）**
**价值评级：★★★☆（"全栈"需求的体现）**

- **React 全家桶**：微聚科技、我想去看看
- **Vue**：超智云、尚瑞天成/心尺科技
- 纯Agent开发岗通常不需要，但如果会前端，"全栈"标签加持，竞争力翻倍

---

## 1️⃣3️⃣ 🔄 Multi-Agent / 多Agent协作

**出现频次：★★★（约30%岗位提及）**
**价值评级：★★★★（技术含金量高，是进阶方向）**

| 具体能力 | 说明 |
|---------|------|
| 多Agent角色拆分 | 不同Agent承担不同角色和职责 |
| 任务分发与协作 | Agent间消息传递、任务分配 |
| A2A通讯 | Agent-to-Agent通信协议 |
| 结果汇总与冲突解决 | 多Agent输出的合并与仲裁 |

**典型岗位：** 平方和投资、深势科技、中科软（均有明确要求）

---

## 1️⃣4️⃣ 🔄 工作流编排（Workflow / OpenCLAW / n8n）

**出现频次：★★★（约25%岗位提及）**
**价值评级：★★★（Agent工程化的关键能力）**

- **OpenCLAW**：轩程科技、职由、平方和投资
- **n8n** / 扣子 / Dify：唯犀互动、奕信科技
- 本质是Agent流程设计 + 条件分支 + 异常处理 + 状态管理

---

## 1️⃣5️⃣ 🧠 模型微调（SFT / Fine-tuning）

**出现频次：★★☆（约20%岗位提及）**
**价值评级：★★★☆（技术壁垒较高，但大部分岗位不需要深度微调）**

- 主要出现在：规模较大的公司（中科软、新智元）或特定场景（芦席智能、唯犀互动）
- 常规Agent岗位更关注**Prompt + RAG + 工具调用**这三个"低成本高收益"手段

# ⚡ 值得关注的差异化亮点技能

以下技能虽然频次不高，但一旦你会，就能在同类候选人中**形成明显差异化**：

1. **Agent评测体系搭建**（只平方和投资、青蓝致远有要求，但价值极高）
2. **MCP协议实践**（2026年新兴，懂的人少）
3. **A2A通讯机制**（平方和投资独有，前瞻性技术）
4. **GraphRAG / 知识图谱**（只有微聚科技、深势科技、中国信通院提及）
5. **AI Coding的TDD实践**（心尺科技、尚瑞天成明确提出）

# 📈 市场趋势总结

1. **Agent是绝对主线**，不是某个公司的偏好，是整个行业的方向
2. **Python + LLM + RAG + LangChain** 是2026年AI应用开发岗的"新四大件"
3. **Vibe Coding / AI Coding工具**从加分项变成必选项，不会用AI写代码就像2016年不会用Git
4. **薪资分化明显**：纯Agent开发大厂岗（字节/BOSS/小红书/上海电商）挂1-3年经验给到30-80K，天津同档15-20K，实习200-400元/天
5. **"全栈+Agent"组合**越来越常见，不会前端也能做Agent但会了更有竞争力
6. **多Agent/MCP/A2A**是2026年新冒头的高价值技能，供需存在缺口
7. **企业AI落地（ERP/OA/知识库/客服）** 是最大的实际需求场景，比炫酷的技术demo更能打动面试官

---

> 📌 **一句话建议：**
> **精通Python + 深度掌握Agent全链路（LLM+RAG+工具调用+LangChain/LangGraph）+ 熟练AI Coding → 这是2026年AI应用开发岗的标准答案。在此基础上，加任意一个差异化亮点（多Agent/MCP/企业系统/评测体系），你就是最有竞争力的候选人。**
