# AI智能体工程师

> 搜索时间: 2026-06-12 08:40:00

## 基本信息

- **公司**: 天津开源信创科技（开元云科技）
- **薪资**: 10-14K
- **经验要求**: 1-3年
- **学历要求**: 本科
- **地点**: 天津西青区360天津科创园
- **招聘联系人**: 宋依然·招聘者


## 岗位职责

1. 智能体架构设计与研发：负责AI智能体的整体架构设计、核心功能研发与迭代优化，包括大模型调用、工具调用、RAG知识库、多轮对话、逻辑规划、记忆模块等核心组件的开发与优化，打造高可用、高扩展的智能体系统。
2. 业务场景智能体落地：深入理解业务需求，针对企业服务、智能客服、办公自动化、行业解决方案等具体业务场景，设计并开发定制化的AI智能体，拆解业务流程，实现业务场景的全流程智能化闭环。
3. 智能体工程化与性能优化：负责智能体系统的工程化落地，包括服务开发、部署、测试、运维全流程工作，优化智能体的响应速度、推理准确率、并发能力与稳定性，确保智能体在生产环境的高效、稳定运行。
4. 工具与生态集成：负责智能体与第三方工具、API、业务系统、知识库的集成开发，包括工具调用链设计、权限管理、数据交互、异常处理等工作，扩展智能体的能力边界与应用场景。
5. 前沿技术跟踪与创新：持续跟踪AI智能体、大模型、多模态、强化学习等领域的前沿技术发展趋势，开展技术预研与创新实践。
6. 跨部门协同与项目交付：与产品、算法、研发、业务等跨部门团队深度协同，推动智能体项目的顺利交付与落地。

## 任职要求

1. 本科及以上学历，计算机相关专业，1-3年 AI 应用开发或智能体相关经验，有至少 1 个智能体项目从开发到上线的完整经验。
2. 扎实的 Python 编程能力，熟悉 FastAPI/Flask 后端开发；熟练使用 LangChain、LlamaIndex、Dify、LangGraph 中至少两个框架，能独立完成对话、RAG、工具调用、记忆模块的开发。
3. 熟悉大模型 API 调用、提示词工程、Function Calling、RAG（含向量库/Embedding/重排序）、多轮对话管理等核心技术。
4. 能独立完成智能体业务模块的开发与上线，具备问题定位与性能调优能力。
5. 良好的工程素养与跨部门沟通能力。

## 公司信息

开元云科技（Open Computing AI），专精特新企业、国家级高新技术企业，聚焦教育科研、金融风控、轨道交通、生物医药等战略场景，与北京大学科创园、百度智能云、阿里云等建立深度合作。

- **公司规模**: 人工智能 0-20人
