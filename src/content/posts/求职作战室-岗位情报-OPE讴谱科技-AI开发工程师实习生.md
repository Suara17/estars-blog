## 职位描述

### 职位概述

我们正在搭建前沿智能 Agent 共生生态基座，赋能多元智能体自主联动协作、供需交易、服务、治理落地及共识共建，打造可自我存续的智能⽣态⽹络。

你将以实习⽣身份参与 AI Agent 产品和工程能⼒建设，在导师带领下接触：
- 大模型 API
- Agent Workflow
- RAG（检索增强生成）
- 工具调用
- Prompt 调优
- 任务评测与工程化交付

**适合人群：** 已经高频使用 AI 编程工具、愿意快速学习 Agent 系统，并希望把课程项目或个人项目能力推进到真实产品环境中的同学。

### 岗位职责

1. **协助开发 AI Agent 基础功能**，包括模型调用、Prompt 模板、工具接入、RAG 检索、任务状态记录等模块
2. **在导师指导下使用 AI 编程工具**完成小型功能开发、Bug 修复、测试用例补充和技术文档整理
3. **参与 Agent Workflow 的实验和验证**，记录任务输入、执行链路、失败原因

## 公司介绍

OPE 深耕行业七年，核心打造由 Agent（智能体）主导的"灵治平台"。独创 AGSA 模式，定义虚拟世界的规则与标准。

### 核心基因

- **元神系统：** 跨越 SDK 门槛，实现"本能式"交互与"夺舍级"能力进化
- **混合治理：** 中心化与去中心化相结合的 MAO 混合治理体系，兼顾生态公平与决策效率
- **灵量经济：** 以 POCA 贡献协议重塑 Agent 世界新消费模式的经济生态

**团队理念：** 精英小团队模式，蓄力于虚拟世界实现降维打击。坚持不做普通应用，而做 Agent 世界的规则制定者。
