## 岗位职责

1. **Agent 设计与搭建**：基于大语言模型，使用 **LangChain、AutoGen** 等框架搭建具备工具调用、多步推理与记忆能力的 AI Agent，落地于**广告投放优化**场景。
2. **Web 管理界面开发**：负责 Agent 配置平台、任务面板与数据看板的前端设计与开发，确保交互流畅、操作直观。
3. **广告策略辅助**：配合产品与算法同学，将 Agent 能力与程序化广告逻辑结合，探索自动出价、素材优选等方向。
4. **技术调研与落地**：持续跟进 **Multi-Agent、GraphRAG**、代码解释器等前沿技术，撰写实验报告，推动能力迭代。

## 任职要求

### 必备
1. **学历专业**：计算机、人工智能、软件工程等相关专业，大三或大四在读，能保证连续实习。
2. **Agent 实践**：熟悉 Agent 搭建与运用，有 **LangChain / LlamaIndex / AutoGen / MetaGPT** 等至少一种框架的实际项目经验，理解 ReAct、工具调用、记忆管理等核心机制。
3. **Web 开发**：具备 Web 前端开发经验，熟悉 HTML/CSS/JavaScript，熟练使用 **React 或 Vue** 全家桶，有可展示的上线作品或课程项目。
4. **编程基础**：**Python** 熟练，具有良好的编码习惯；了解 RESTful API 设计，能与后端协作。
5. **软技能**：主动性强，善于沟通，具备较好的逻辑思维与学习能力。

### 加分项
- 了解线上广告生态，如 **RTB、DSP、SSP**、信息流广告或广告投放系统的基本流程。
- 有 LLM 应用全栈开发经验（**FastAPI / Flask** 后端 + 前端），能独立跑通完整项目。
- 熟悉向量数据库、**Prompt Engineering**、RAG 管道优化。
- 参与过 AI 竞赛或开源项目，有 GitHub 个人主页或技术博客。
